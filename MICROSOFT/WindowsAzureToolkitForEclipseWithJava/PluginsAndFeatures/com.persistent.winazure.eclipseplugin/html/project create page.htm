Create Azure Deployment Project

Using this wizard you can create an Azure Deployment Project.

Step 1: Specify project name of the project in specified text box. It is similar to specifying java project name in eclipse.

Step 2: Check “Use default location” checkbox if you want to use default location or specify your own location to create the project. Default location will be inside your current workspace.

Step 3: You can add the project being created to the existing working sets of eclipse or can create your own working set. This is optional.

Step 4: By default there is one role (WorkerRole1) added to the new project, the VM size of this role will be Small and by default 1 instance is added. To Add \ Edit \ Remove role, you can click on respective buttons.

Step 5: Click on finish button to create new project at specified location.

For more information: http://java.interoperabilitybridges.com/cloud
