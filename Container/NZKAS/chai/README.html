Chai is a BDD / TDD assertion library for node and the browser that can be delightfully paired with any javascript testing framework.

For more information or to download plugins, view the documentation.

Related Projects

  • chaijs / assertion-error: Custom Error constructor thrown upon an assertion failing.
  • chaijs / deep-eql: Improved deep equality testing for Node.js and the browser.
