## Getting Started

Because this website is a [Gatsby](https://www.gatsbyjs.org/) project, local development is as easy as:

```bash
git clone https://github.com/newrelic/developer-website.git
yarn install
yarn start
```

Find the detailed instructions for getting started in the project [README](https://github.com/newrelic/developer-website/blob/develop/README.md)
