## Features

The New Relic Handbook allows you to create an onboarding guide for your teammates, helping them get set up for success with New Relic with specific guidance related to your New Relic installation. All of this backed up by links back to New Relic Documentation.
Fork this handbook for your organisation, and let us know how you are using it. 

Out of the box this Handbook comes with information on getting set up with: 

- APM
- Browser 
- Infrastructure
- Synthetics
- New Relic One Programmability
- APIs
- Etc.. 

With a well defined handbook, adjusted to match how your organisation works with New Relic, you can get your colleagues up and running very quickly.
