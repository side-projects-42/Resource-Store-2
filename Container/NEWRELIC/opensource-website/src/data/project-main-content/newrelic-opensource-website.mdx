Fork or contribute code to this website. Very _meta_.

## Getting Started

Because this website is a [Gatsby](https://www.gatsbyjs.org/) project, local development is as easy as:

```bash
git clone https://github.com/newrelic/opensource-website.git
yarn install
yarn start
```

Go to the project's [README](https://github.com/newrelic/opensource-website#readme) for setup and usage details.

## Features

- Register new/update projects via a [documented PR process](https://github.com/newrelic/opensource-website#contributing-project-data)
- Automatically refreshes [projects statistics](https://github.com/newrelic/opensource-website/tree/master/src/data/project-stats) via GitHub Action automation
- Leverages [MDX](https://mdxjs.com/) for hybrid React/Markdown pages
