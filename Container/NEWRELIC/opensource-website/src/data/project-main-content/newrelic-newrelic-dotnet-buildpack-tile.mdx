## Getting Started

Go to the project's [README](https://github.com/newrelic/newrelic-dotnet-buildpack-tile#readme) for setup and usage details.

This repository contains [New Relic Dotnet Extension Buildpack Tile for VMware Tanzu (PCF)](https://network.pivotal.io/products/new-relic-dotnet-buildpack/). Please refer to detailed [overview](https://docs.pivotal.io/partners/new-relic-dotnet-buildpack/index.html), [installation and configuration instructions](https://docs.pivotal.io/partners/new-relic-dotnet-buildpack/installing.html), [usage](https://docs.pivotal.io/partners/new-relic-dotnet-buildpack/using.html), and [release notes](https://docs.pivotal.io/partners/new-relic-dotnet-buildpack/release-notes.html) to bind your application with New Relic Dotnet agent for monitoring. You can download the buildpack from [VMware Tanzu](https://tanzu.vmware.com/tanzu).
