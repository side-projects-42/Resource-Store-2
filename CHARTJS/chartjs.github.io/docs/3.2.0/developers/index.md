<a href="#browser-support" class="header-anchor">#</a> Browser support
----------------------------------------------------------------------

All modern and up-to-date browsers are supported, including, but not limited to:

Chrome Edge Firefox Safari

As of version 3, we have dropped Internet Explorer 11 support.

Browser support for the canvas element is available in all modern & major mobile browsers. [CanIUse <span class="sr-only">(opens new window)</span>](https://caniuse.com/#feat=canvas)

Run `npx browserslist` at the root of the [codebase <span class="sr-only">(opens new window)</span>](https://github.com/chartjs/Chart.js) to get a list of supported browsers.

Thanks to [BrowserStack <span class="sr-only">(opens new window)</span>](https://browserstack.com) for allowing our team to test on thousands of browsers.

<a href="#previous-versions" class="header-anchor">#</a> Previous versions
--------------------------------------------------------------------------

To migrate from version 2 to version 3, please see [the v3 migration guide](../getting-started/v3-migration).

Version 3 has a largely different API than earlier versions.

Most earlier version options have current equivalents or are the same.

Please note - documentation for previous versions are available online or in the GitHub repo.
